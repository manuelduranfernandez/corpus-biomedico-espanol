Correspondencia          a :   Hay una serie de programas novedosos que          son convenientes de conocer.    Uno de ellos es el Picasa. Este programa,          que se puede descargar de http://www.          picasa.com/download/index.php, permite organizar todas las fotografías          que se encuentren ubicadas dentro del computador de una manera absolutamente          automática y eficiente. Se debe ejecutar el programa e indicar          que realice una búsqueda completa (figura 1).              

   Figura 1.     Este proceso, que puede demorar unos minutos,          permitirá que el programa realice un índice de todas las          fotografías y que permita un acceso instantáneo. Permite          editarlas, con una serie de funciones que cualquiera, sin experiencia          en programas gráficos lo puede hacer, realizar automáticamente          respaldos, importar nuevas fotografías, etc. En resumen, este programa          pone orden en un sistema en que en general es muy desordenado. Se pueden          identificar grupos de fotos y asignarles fechas y nombres que las identifiquen          con facilidad (figura 2).    

   Figura 2.     Otro programa muy útil es el Winrar.          Se puede descargar desde http://www.rarlab.com/download.htm.          Permite comprimir y descomprimir archivos. Aunque el más conocido          es WinZip, este programa es más completo, ya que incluye una serie          de otros sistemas populares de compresión, tal como el RAR. Es          muy sencillo de instalar y de utilizar. Se debe aceptar todas las funciones          asociadas a archivos, excepto el de ZIP que es manejado automáticamente          por Windows (figuras 3 y 4).     

   Figura 3.     

   Figura 4.     En forma relativamente sencilla se pueden          descomprimir los archivos extrayéndolos o se pueden comprimir con          el objeto de poder enviarlos por e-mail o achicarlos para poder grabarlos          en un pendrive. Una de las virtudes que tiene este programa es que se          puede definir el tamaño del archivo comprimido que se desea. El          programa automáticamente separa los archivos y crea varios segmentos          separados de los archivos originales con el tope del tamaño predesignado          (figura 5):    

   Figura 5.     Con el botón derecho del Mouse          se marca el archivo que se quiere comprimir. Se indica "add to archive",          y aparece una nueva pantalla en que en la sección "split to          volumen" se debe indicar el tamaño que se desea (en este caso          archivos de 500.000 bytes cada uno). Posteriormente, quedaran en el mismo          archivo donde se crearon. En este caso, hay 8 archivos RAR de 500.000          bytes. Para descomprimirlo, se hace doble clic en el último de          los archivos (en este caso el 8) y automáticamente el programa          los junta en uno solo. Es muy útil cuando se desea enviar vía          email y en muchas casillas aún el límite de envío          está limitado (figuras 6 y 7).    

   Figura 6.      

   Figura 7.    Aunque no es un programa, hay que mencionar          las nuevas casillas de correo de alta capacidad. Aunque Yahoo cambió          sus casillas a 1 Gigabyte de capacidad, una de las de mayor capacidad          es GMail (perteneciente a Google), que tiene una capacidad de 2 Gigabytes.
